Projects

Current Main Project

Right now, my biggest "project", so to speak, is to finish up the YWebCA, a web development program offered by the YWCA of Madison, Wisconsin. The program is an 11 week boot camp for web development, were students learn HTML, CSS, JavaScript, and jQuery while learning the basics of creating a website from the ground up.


Newest Project

Recently, myself and friends of mine, with a collaboration with the school districts of Madison, Middleton, and Verona, and with help of the Centro Hispano of Madison, have created a new project that we call "La [Re]Generacíon", which is a Youth-Led Participatory Action Research, in which Latin@ students will be able to conduct surveys and make thier own research on how the Latino community here in Madison is doing, diagnose what problems are and are persisting in our community, obtain data and information, being able to present that data in any way possible, through art, info-graphs, collective essays, and to create solutions that we are then able to apply and achieve later on. It is a very early and time consuming project, but in the end it will be much well worth the effort to do so.
